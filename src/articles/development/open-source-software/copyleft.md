# 카피레프트

카피레프트는 리처드 스톨만(Richard M. Stallman)이 창시하였으며, 저작권(copyright)에 반대되는 개념입니다. 저작권을 기반으로 한 사용 제한 대신, 정보의 공유를 목적으로 합니다.

저작권에 의한 정보의 독점을 막고, 자유로운 정보 공유를 달성하는 것이 카피레프트의 목적입니다.
